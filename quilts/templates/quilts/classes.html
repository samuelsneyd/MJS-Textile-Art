Classes

I make realistic collages using a raw edge, fused fabric technique, and by participating in my classes, I will teach you to do the same.

I specialise in portraits, nudes and people, although the techniques I teach can equally well be used for animals, landscapes, still life etc.

I currently offer three classes (click on the link to get more information):

  1. A 1-day beginners' class: Value and Shadows.
  2. A 2-day intermediate class: Using Value for Realism: Make a Realistic Hand.
  3. An advanced 4-day class: Using Value for Realism: Make a Realistic, Monochromatic Portrait.

I am also developing another intermediate level class Using Value for Realism: Make a Realistic Rhino so that people have more choice of classes before participating in my advanced portrait class. These classes are face-to-face in New Zealand only at the minute.

Please email me to inquire about classes.
